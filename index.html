What this version changes

This redesign shifts the site away from a concept-heavy visual style and toward a cleaner engineering portfolio: stronger hierarchy, more readable text, calmer color use, and more consistent page structure.

Presentation

Clearer first impression

The homepage explains who you are, what you work on, and where a visitor should go next without relying on decorative effects.

Consistency

One visual system

Navigation, cards, spacing, typography, and buttons now follow one shared pattern across all main pages.

Readability

More professional tone

The design uses lighter surfaces, darker body text, and less visual noise so the work itself carries more of the page.

Current interests

Technical areas I want to develop further

  • Liquid propulsion systems and supporting aerospace analysis work.
  • Structural mechanics, load paths, and design validation.
  • Computational tools that support engineering decision-making.
  • CAD-driven design workflows from concept to prototype.
Professional direction

A portfolio can still be in progress and look intentional.

The issue is not that some work is unfinished. The issue is when unfinished work is presented with placeholder layouts or generic filler. This redesign keeps the site honest about being in progress, but gives it a stronger editorial standard.
